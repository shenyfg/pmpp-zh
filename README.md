# pmpp-zh

Programming Massively Parallel Processors Fourth Edition（大规模并行处理器设计）完整解答

## 介绍

这个项目作为笔者学习《大规模并行处理器设计》第4版这本书的笔记。
网上可以找到英文版题目解答，但是没有中文版的，且排版较乱（在题目下面就有解答，无法用来参考练习），这个仓库旨在解决这个问题。

题目都在`exercises`里，答案都在`solutions`里。

需要注意的是，所有`exercises`都由英文原题翻译而来，所有`solutions`都是作者自己学习过程中的习作。如果有错误，欢迎勘误。
